# Kahwe Ti Kath ☕

A modern, elegant website for a specialty coffee shop featuring stories and soul in every sip.

## About

**Kahwe ti Kath** is a beautifully designed coffee shop website that showcases the brand's culture and offerings. The site features a warm, inviting aesthetic with an elegant design that reflects the coffee shop's passion for quality and community.

## Features

- **Responsive Design** - Optimized for desktop and mobile devices
- **Smooth Navigation** - Seamless scrolling experience with a sticky navbar
- **Sections Included:**
  - Home - Hero section with brand messaging
  - About - Coffee shop story and mission
  - Menu - Coffee and beverage offerings
  - Gallery - Visual showcase of the space
  - Visit - Location and hours
  - Contact - Reservation and contact information

## Tech Stack

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with glassmorphism effects
- **Google Fonts** - Elegant Cormorant Garamond and clean Montserrat typefaces

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through the sections using the navbar
3. Click "RESERVE A TABLE" to book a spot
